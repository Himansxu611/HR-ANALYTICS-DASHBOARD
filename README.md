HR Analytics Dashboard – Power BI

This project analyzes employee data to understand workforce trends and identify attrition patterns. The dashboard was built using Power BI to visualize key HR metrics and support data-driven decision-making.

Tools Used
- Power BI
- Power Query
- DAX

 Key KPIs
- Employee Count
- Attrition Rate
- Department-wise Attrition
- Salary Distribution
- Job Role Analysis

 Insights
The dashboard helps HR teams monitor employee attrition, analyze department performance, and identify factors contributing to employee turnover.
